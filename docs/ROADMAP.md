# Roadmap

Future plans and features for GitHub Daily Activity.

## Current Version: 1.0.0

### Completed Features

- [x] Automatic boot detection
- [x] System information collection
- [x] SSH-based authentication
- [x] Internet retry system
- [x] Systemd integration
- [x] Auto branch detection
- [x] Safe git operations
- [x] Comprehensive logging
- [x] Multiple command modes
- [x] Configuration file
- [x] Installation scripts
- [x] Complete documentation
- [x] ShellCheck CI
- [x] GitHub templates

## Planned Features

### Version 1.1.0 (Short-term)

- [ ] Webhook notifications (Telegram/Discord/Slack)
- [ ] Custom boot entry templates
- [ ] Log rotation built-in
- [ ] Status email reports

### Version 1.2.0 (Medium-term)

- [ ] GUI configuration tool
- [ ] Statistics dashboard
- [ ] Multiple repository support
- [ ] Custom commit messages
- [ ] Boot time patterns analysis

### Version 2.0.0 (Long-term)

- [ ] Python port (cross-platform)
- [ ] Web interface
- [ ] API for external tools
- [ ] Plugin system
- [ ] Cloud sync options
